## 📝 Descrição do Projeto
Este repositório contém a implementação prática de tipos abstratos de dados (TAD) focados em listas sequenciais (arrays). O objetivo é demonstrar o funcionamento interno de operações de inserção, remoção, redimensionamento dinâmico e fusão de dados, analisando o custo computacional dessas operações.

## 🚀 Exercícios Implementados
1. **Deslocamento para Inserção:** Simulação do custo $O(n)$ ao deslocar elementos para a direita para inserir um novo valor em um índice específico.
2. **Redimensionamento Dinâmico:** Implementação de um vetor que detecta quando está cheio e dobra sua capacidade automaticamente.
3. **Exclusão com Continuidade:** Remoção de elementos utilizando *shift left* (deslocamento para a esquerda) para evitar "buracos" no array.
4. **Remoção de Duplicatas:** Algoritmo que identifica e remove valores repetidos, mantendo a coleção organizada.
5. **Merge de Arrays Ordenados:** Fusão de dois arrays já ordenados em um terceiro, percorrendo ambos simultaneamente para manter a ordenação final.
